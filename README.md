# Thai Dummy

A browser implementation of Thai Rummy for one player against a computer opponent.

## Features

- Two-player 13-card deal
- Stock draw and multi-card discard-pile pickup
- Sets, same-suit runs, and extensions to table melds
- Computer opponent
- Thai Dummy point values, going-out bonus, doubling, and penalties
- Multi-hand match scoring
- Responsive tabletop interface and in-game rules

The rules are based on [Ben Lynn's Thai Rummy notes](https://benlynn.blogspot.com/2008/05/thai-rummy_16.html).

## Development

```bash
pnpm install
pnpm dev

```
